# Windsurf-Tool

Windsurf 账号管理工具 — 批量注册、一键切号、批量绑卡、积分查询、账号导入导出。

适用于 **Windows** / **macOS** | 完全开源 | 本地运行

## 功能特性

- **批量注册** — 自动完成邮箱注册、验证码填写，支持并行注册
- **一键切号** — 快速切换不同 Windsurf 账号
- **批量绑卡** — 自动获取 Stripe 试用链接、填写支付表单、选择支付宝支付，滑动窗口并行处理
- **积分查询** — 一键查询账号剩余积分
- **账号导入/导出** — 支持批量导入导出，可筛选仅导出未导出过的账号
- **订阅状态检测** — 自动检测账号订阅类型（Free / Trial / Pro）

## 使用方式

### 1. 安装依赖

```bash
npm install
```

### 2. 启动

```bash
npm run start
```

### 3. 系统设置

首次使用请进入「系统设置」配置以下内容：

- **Chrome 路径** — 如果自动检测失败，需手动指定本地 Chrome 浏览器路径
- **代理设置** — 如需代理访问，填写代理地址（如 `http://127.0.0.1:7897`）
- **网络中转配置** — 国内用户需要配置 Cloudflare Worker 中转地址，用于访问 Firebase API

### 4. 批量注册

1. 进入「批量注册」页面
2. 配置邮箱域名、密码规则、注册数量、并行数
3. 点击「开始注册」，浏览器自动完成注册流程

### 5. 批量绑卡

1. 进入「绑卡」页面
2. 配置支付方式（支付宝）和账单信息（姓名、地址等）
3. 设置并行数量（建议 1-3）
4. 点击「批量绑卡」，工具自动完成：
   - 登录账号 → 获取试用链接 → 打开 Stripe 页面 → 选择支付宝 → 填写表单 → 点击订阅
5. 弹出支付宝扫码页面时，手动扫码完成支付

### 6. 账号导出

- 支持导出为 JSON / CSV / TXT 格式
- 可勾选「仅未导出过」只导出新账号

## 注意事项

- **Chrome 浏览器** — 必须安装 Google Chrome，工具使用 Puppeteer 驱动 Chrome 完成自动化操作
- **网络环境** — 注册和绑卡过程需要能访问 windsurf.com 和 stripe.com，国内用户建议配置代理
- **Cloudflare Turnstile** — 绑卡过程中可能遇到 Turnstile 验证，工具会自动处理，如验证失败请稍后重试或更换代理 IP
- **并行数量** — 批量绑卡建议并行数不超过 3，过多可能触发风控
- **支付宝扫码** — 绑卡最后一步需要手动扫码支付，请留意弹出的浏览器窗口
- **数据安全** — 所有账号数据仅存储在本地 `%APPDATA%/windsurf-tool/`（Windows）或 `~/Library/Application Support/windsurf-tool/`（macOS），不上传任何数据
- **取消操作** — 批量绑卡过程中可随时点击「取消」，所有浏览器窗口会立即关闭

## 自定义 Worker

如果默认中转域名失效，可使用仓库中的 `cloudflare-worker/firebase-auth-proxy.js` 自行部署：

1. 在 Cloudflare Workers 创建新 Worker
2. 粘贴 `cloudflare-worker/firebase-auth-proxy.js` 内容并发布
3. 在软件「系统设置」→「网络中转配置」填入 Worker 地址
4. 重启软件生效
